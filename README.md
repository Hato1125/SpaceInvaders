# SpaceInvaders
現在、開発中です。

## 操作
| 操作 | キー |
|:--|:--|
|コイン挿入|`P` / `PGUP`|
|サービスクレジット|`S`|
|決定|`SPACE`|
|左右移動|`← →`|
|ビーム発射|`SPACE`|

## DEBUGモードのみ有効な操作
| 操作 | キー |
|:--|:--|
|最初のシーンに戻る|`HOME`|
|GCを起こす|`G`|
|データベースにテスト用データを追加|`D`|
|データベースのデータをすべてクリア|`DELETE`|
